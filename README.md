# Next-Gen Student Ecosystem: Rule-Based Academic Risk Analytics

## Overview

This project is a Python Django-based academic analytics system that identifies students at risk using rule-based logic.

## Features

* Student performance analysis
* Attendance and behavior tracking
* Risk classification (Low, Medium, High)
* Dashboard visualization
* Keyword-based chatbot

## Tech Stack

* Python
* Django
* SQLite
* Pandas
* HTML, CSS, Bootstrap

## How to Run

1. Install dependencies:
   pip install -r requirements.txt
2. Run:
   python manage.py runserver

## Note

This is a rule-based system and does not use machine learning.
